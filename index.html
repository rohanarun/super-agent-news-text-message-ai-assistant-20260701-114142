How a text message becomes action

1. Send a plain text

Text Super like you would a human assistant: “open twitter on device 2” or “run the marketing play on all devices.” This mirrors documented text-message control in Super’s own docs (app.getsupers.com).

2. Super routes it

Your message is routed to the correct cloud or connected computer using the same account and device rules described in SuperPowers documentation.

3. Real computer use

Super clicks, types, scrolls, and navigates browsers and apps — not just APIs. This is where most SMS bots stop and Super keeps going.

4. Reuse with computer-use cache

Successful runs are stored in a reusable computer-use cache, so the next time you text the same request it’s faster and cheaper to run.

Why text instead of another app?

Zero context switching

SMS is already your fastest interface. Texting an assistant removes dashboards, menus, and logins.

Great for follow‑ups

Send “do this again tomorrow” or “run that weekly.” Super can schedule or replay actions you already approved.

Built for repetition

Because Super reuses a computer-use cache, repeated text‑driven workflows don’t reset to zero every time.
